GDPR & Dataskydd

Senast uppdaterad: April 2026

NordAssist är utformat med dataskydd som standard. Vi samlar in minsta möjliga data, lagrar allt inom EU och gör det enkelt för dig att hantera dina kunders uppgifter i enlighet med GDPR.

Personuppgiftsansvar

NordAssist agerar som personuppgiftsbiträde åt dig som kund när vår widget samlar in data från dina webbplatsbesökare. Du som kund är personuppgiftsansvarig för dessa uppgifter.

För data som du lämnar till NordAssist vid registrering och nyttjande av tjänsten är NordAssist personuppgiftsansvarig.

Datalagring

  • All data lagras inom EU/EES hos Supabase (Irland, ISO 27001-certifierat)
  • Ingen data säljs eller delas med tredje part för marknadsföring
  • Konversationsdata sparas i 90 dagar, lead-data tills du tar bort den
  • Vid avslut av konto raderas all data inom 30 dagar

Dina skyldigheter som kund

Som kund ansvarar du för att:

  • Informera dina webbplatsbesökare om att NordAssist används via din integritetspolicy
  • Ha rättslig grund för insamling av personuppgifter via widgeten
  • Hantera förfrågningar om radering och tillgång från dina besökare

Personuppgiftsbiträdesavtal

NordAssist erbjuder ett personuppgiftsbiträdesavtal (DPA) för kunder som kräver det. Kontakta oss på hej@nordassist.ai för att begära ett signerat avtal.
